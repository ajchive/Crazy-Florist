# The Crazy Florist

## Setup Instructions

### 1. Install XAMPP
Download and install XAMPP.

### 2. Move Project
Copy the CrazyFlorist folder into:
C:\xampp\htdocs\

### 3. Start Server
Open XAMPP Control Panel and start:
- Apache
- MySQL

### 4. Import Database
1. Open phpMyAdmin (http://localhost/phpmyadmin)
2. Create a new database: crazy_florist
3. Click Import
4. Upload the file: database.sql

### 5. Run Website
Open browser:
http://localhost/CrazyFlorist/
